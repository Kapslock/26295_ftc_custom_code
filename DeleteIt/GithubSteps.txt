Configure Git identity (once per machine):

git config --global user.name "Their Name"
git config --global user.email "their@email.com"


Clone your fork directly:

git clone https://github.com/knishant546/FtcRobotController.git
cd FtcRobotController


Make a new branch, commit, and push with upstream:

git checkout -b feature/my-work
git add .
git commit -m "my changes"
git push -u origin feature/my-work
